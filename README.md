📌 Project Overview

Clevision is a modern, responsive single-page website built as a frontend development task. The webpage consists of three main sections: Header, Hero, and Footer. The design is fully responsive and adapts seamlessly across desktop, tablet, and mobile devices.

Features
Responsive Navigation Bar
Mobile Hamburger Menu
Attractive Hero Section
Call-to-Action Buttons
Responsive Footer
Clean UI Design
Smooth User Experience Across Devices
🛠️ Tech Stack

The project was developed using:

HTML5 – Structure and content
Tailwind CSS – Styling and responsive design
JavaScript (ES6) – Interactive functionality and mobile navigation
